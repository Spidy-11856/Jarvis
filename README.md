## ✨ Features

### 🔍 Web Search & Research
- Search the internet using **DuckDuckGo** & **Wikipedia** (free, no API key)
- Deep research mode with multi-source analysis
- Page content fetching and summarization

### 💻 Code Generation
- Generate code in **any programming language**
- Python, JavaScript, HTML, CSS, SQL, Bash, and more
- Full project scaffolding

### 🎤 Speech-to-Text
- Built-in voice recognition (Web Speech API — free)
- Voice mode with animated orb UI
- Text-to-speech responses

### 🔴 Beautiful UI
- Stunning red glow ring with motion animations
- Animated wave responses
- Clean ChatGPT-style interface
- Dark mode by default

### ✏️ Self-Editing
- JARVIS can modify its own source code
- Change colors, fonts, behavior on command
- Live CSS variable updates

### 📁 File Management
- Create and download files
- Multiple file format support
- Built-in file browser

### 🧠 Local Intelligence
- Math calculations
- Text summarization
- Conversation memory
- Slash commands

## 🚀 Quick Start

### Option 1: GitHub Pages (Recommended)
1. Fork this repository
2. Go to **Settings → Pages**
3. Set Source to **main branch**
4. Visit `https://yourusername.github.io/jarvis`

### Option 2: Local
1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/jarvis.git
   cd jarvis
   ```
2. Open `index.html` in your browser (Chrome/Edge recommended)

### Option 3: Any Web Server
Just upload all files to any web hosting — it's pure HTML/CSS/JS!

## 📖 Commands

| Command | Description |
|---------|-------------|
| `/search [query]` | Search the web |
| `/research [topic]` | Deep multi-source research |
| `/code [request]` | Generate code |
| `/create [filename]` | Create a file |
| `/calculate [expr]` | Math calculation |
| `/summarize [text]` | Summarize text |
| `/self-edit` | Enter self-edit mode |
| `/export` | Export conversation |
| `/clear` | Clear history |
| `/help` | Show all commands |

You can also use **natural language** — just talk normally!

---

## 🎤 Voice Mode

1. Click the **🎤 Voice** button in the top bar
2. Click the microphone icon
3. Speak your command
4. JARVIS will respond with text and speech

> **Note:** Speech-to-text uses the browser's built-in Web Speech API. Works best in Chrome & Edge.
